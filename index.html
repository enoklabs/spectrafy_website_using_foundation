What is Spectrafy?

Spectrafy is a desktop Adobe Air Application, that allows users to find a large range of color palettes used for finding color inspiration. The color themes found in Spectrafy are suitable for any modern and classic designer.

Features

  •     Unlimited Color Schemes
  •   Copy color Hex Code
  •    Save color swatches
  •    Links to Adobe Kuler
  •   PC/Mac/Linux Compatible
  •   It's Completely Free!
More Info

Spectrafy is a free downloadable Abode Air Application that uses the Adobe Kuler API to provide color palettes to the end user. Spectrafy will simplify the process of exploring and using color palettes that are widely used by designers of the digital mediums.

Inside the application the user will be able to find color palettes based on keywords of colors, moods, and words in general. Then the application will serve the combination of colors pre-made by users of the main Kuler's cloud based application from Adobe.
